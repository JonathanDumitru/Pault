# macOS Screenshot Capture Plan

## Target shots

Capture these six screens in the same theme and data set:

1. Library overview with sidebar filters.
2. Prompt detail editor with real prompt content.
3. Menu bar popover open.
4. Global launcher search results.
5. Copy and paste action row.
6. Favorites + archived views.

## Capture rules

- Use one language locale for all images.
- Keep window size and zoom consistent.
- Remove personal data and API keys from prompt text.
- Use realistic sample prompts.
- Export PNGs.

## Recommended naming

- `01-library-overview.png`
- `02-editor-detail.png`
- `03-menu-bar-popover.png`
- `04-global-launcher.png`
- `05-quick-paste.png`
- `06-favorites-archive.png`

## macOS capture command

Use the built-in command after selecting the app window:

```bash
screencapture -i ~/Desktop/01-library-overview.png
```

Repeat for each shot and upload all required size sets in App Store Connect.
