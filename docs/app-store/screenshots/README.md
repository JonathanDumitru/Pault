# Screenshot Inputs For PDF Visual Walkthrough

Place PNG screenshots in this folder to populate the stylized User Guide PDF screenshot pages.

## Preferred files

- `01-library-overview.png`
- `02-editor-detail.png`
- `03-menu-bar-popover.png`
- `04-global-launcher.png`
- `05-quick-paste.png`

## Page mapping in the PDF

- Main Window Workflow page:
  - uses `01-library-overview.png` or `02-editor-detail.png`
- Menu Bar Quick Access page:
  - uses `03-menu-bar-popover.png`
- Hotkey Launcher page:
  - uses `04-global-launcher.png` or `05-quick-paste.png`

If a file is missing, the PDF renders a branded placeholder panel and lists the expected filenames.
